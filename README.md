# Al Khaliq Backend Docs

Docusaurus site for the Al Khaliq music streaming backend.

## Develop

```bash
npm install
npm run start
```

Opens at <http://localhost:3000>.

## Build

```bash
npm run build
```

Output in `./build`.

## Deploy

Pushing to `main` triggers `.github/workflows/deploy.yml`, which publishes `./build` to GitHub Pages.

The workflow needs a `PERSONAL_TOKEN` secret (a classic PAT with `repo` scope for the same org) set in repo settings.

## Stack

- Docusaurus 3.9.2, React 19, TypeScript 5.6
- Classic preset, autogenerated sidebar from folder structure + `_category_.json` metadata
- No blog, no i18n, no versioning (added when needed)

## Writing docs

- `docs/<category>/_category_.json` controls category label, position, and the generated-index description.
- Markdown files get auto-ordered inside each category unless they carry `sidebar_position:` frontmatter.
- Link to other docs with relative paths: `[Tokens](./tokens.md)`.
- Code blocks support Prism syntax highlighting — specify the language fence (` ```ts `, ` ```bash `, etc.).

Source of truth for implementation lives in the backend repo. This site explains **design, architecture, and runbooks** — the *why* alongside the *how*.
